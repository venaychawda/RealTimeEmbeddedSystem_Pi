# Day 16: V4L2 — USB Camera on Linux

## Goal
Query camera capabilities and capture frames using command-line tools.

## Run
```bash
./capture_frames.sh
```

## Expected Output
- Camera capabilities listed (driver, resolution, controls)
- Supported formats (YUYV, MJPG) with resolutions and fps
- 10 JPEG files created (frame_001.jpg to frame_010.jpg), each 20–100 KB
